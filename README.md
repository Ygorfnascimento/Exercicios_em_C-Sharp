<h3 id="exercicios" align="center">🧾 Exercícios </h3>

<h3 align="center">🟢 Atividade 1 - Encontrar o Maior e o Menor Valor</h3>
<p align="center"><strong>Objetivo:</strong> Permitir que o usuário insira três números e descubra qual é o maior e o menor entre eles.</p>

<p align="center"><strong>Como funciona:</strong></p>
<ul align="left">
  <li>O usuário preenche três campos com valores numéricos.</li>
  <li>Ao clicar no botão "Calcular", o programa identifica o maior e o menor valor.</li>
  <li>O resultado é exibido em uma <code>MessageBox</code>.</li>
</ul>

<p align="center"><strong>Exemplo de uso:</strong></p>

```text
  Digite os números: 5, 8, 2.
  Clique em "Calcular".
  O programa exibirá "Maior valor: 8" e "Menor valor: 2".
```

---

<h3 align="center">🟠 Atividade 2 - Calcular o Peso em Diferentes Planetas</h3>
<p align="center"><strong>Objetivo:</strong> Calcular o peso do usuário em diferentes planetas com base na gravidade de cada um.</p>

<p align="center"><strong>Como funciona:</strong></p>
<ul align="left">
  <li>O usuário informa seu peso na Terra.</li>
  <li>Seleciona um planeta da lista (Mercúrio, Marte, Urano, Vênus, Saturno, Júpiter).</li>
  <li>Se o valor informado não for numérico ou nenhum planeta for selecionado, o programa mostra uma mensagem de erro.</li>
  <li>O programa multiplica o peso informado pelo fator de gravidade do planeta escolhido (ex: Mercúrio = 0.37, Marte = 0.38).</li>
  <li>O peso ajustado é mostrado em uma <code>MessageBox</code>.</li>
</ul>

<p align="center"><strong>Exemplo de uso:</strong></p>

```text
  Informe seu peso na Terra: 70 kg.
  Selecione o planeta Marte.
  O programa exibirá "Seu peso em Marte é 26.6 kg".
```

<h3 align="center">🔵 Atividade 3 - Verificar o Resultado de Jogos de Futebol</h3>
<p align="center"><strong>Objetivo:</strong> Identificar o vencedor (ou empate) entre dois times com base nos gols inseridos.</p>

<p align="center"><strong>Como funciona:</strong></p>
<ul align="left">
  <li>O usuário insere os nomes dos times e os gols de cada um.</li>
  <li>O programa compara os valores informados.</li>
  <li>Exibe quem venceu ou se houve empate, usando <code>Labels</code> na interface.</li>
</ul>

<p align="center"><strong>Exemplo de uso:</strong></p>

```text
Insira o nome do time: "Time A" e "Time B".
Insira os gols: "3" e "2".
O programa exibirá "Time A venceu".
```
